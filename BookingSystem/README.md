# Centrum Air Overbooking Project

## Description
This project demonstrates overbooking in a flight booking system using C and multithreading.

## Files
- `backend/no_lock.c` – Booking without mutex (can cause overbooking)
- `backend/with_lock.c` – Booking with mutex (prevents overbooking)
- `report.docx` – Explanation of the project and OS concepts
